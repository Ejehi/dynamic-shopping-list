# A dynamic shopping list

A dynamic shopping list that allows you to add items using a form input and a button

## How To Use This Shopping List

After you type an item in the input field and click the button or press the Enter key, the following should happen:

- The item should appear in the list.
- Each item should have a button next to it that removes the item from the list when clicked.
- The input fields should be cleared and focused, ready for the next item entry.
